# 設備設計費・一般業務人時間数 計算アプリ

国土交通省「官庁施設の設計業務等積算要領（令和6年改定）」別表1-1の新築設計業務「設計・設備」欄の係数を用いて、設備設計の一般業務人・時間数と概算見積金額を計算する簡易Webアプリです。

公開URL: https://fdoyoshikazuhora-sys.github.io/equipment-design-calculator/

## 主な機能

- 第一号から第十二号までの建築物類型に対応
- 第1類 / 第2類の区分選択
- 延べ床面積による設備設計一般業務人・時間数の算定
- 単一用途モード、複合用途モード
- 設計・設備の複合化係数
- 主用途扱いとの比較
- 工種別内訳
  - 電気設備
  - 空調換気設備
  - 給排水衛生設備
  - 昇降機等
  - 共通・調整
- 技師C基準日額から時間換算単価を計算
- 諸経費、技術料等経費、特別経費、消費税の概算計算
- 概算見積書プレビュー、見積書印刷
- 入力条件のブラウザ内保存
- JSON書き出し・読み込み

## 注意

このアプリは簡易計算用です。正式な積算、契約、見積、発注判断では、最新の要領、契約範囲、追加業務、特別経費、適用年度の技術者単価を確認してください。

工種別内訳は、国交省の設計・設備係数により算出した設備一式の業務人・時間数を、配分率で按分した概算です。国交省係数自体が電気・空調・衛生等の工種別係数を示しているわけではありません。

## プライバシー

入力内容、保存済み案件、JSONファイル、見積内容はブラウザ内で処理されます。閲覧状況の把握のため、Cloudflare Web Analyticsによるアクセス解析を利用しています。
